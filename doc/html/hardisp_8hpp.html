Detailed Description

Additional C++ functions and definitions implementing the IERS 2010 standards; This header file accompanies the hardisp software.

Note
  1. Original FORTRAN software can be found at http://maia.usno.navy.mil/conv2010/software.html
  2. The compilation flag QUICK_EXIT can be set to implement a quick version but reduced accuracy option. See th readme. file for details.
  3. For more information, see the iers2010.hpp file at the root directory.
