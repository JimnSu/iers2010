Intorduction

In order to use the software package (gpt2.cpp), you will need an (ascii) file called gpt2_5.grd, which provides a grid and is read by some function. This file is distributed by the IERS and can be found at its website (). To download the files, you can use a web-browser or just type: 

bash$wget ftp://maia.usno.navy.mil/conv2010/convupdt/chapter9/gpt2_5.grd
