Testing the C++ Implementation

Once you have sucessefuly installed the libiers10++ library (see Chapter 2, Installation), you should verify the results using the program test.cpp. The source code for this program, can be located in the test directory.

Procedure 4.2. Making C++ test.e

  1. Navigate to the test folder; there should be a file named test.cpp.

  2. Compile the program, linking with the libiers10++ library. To do this, execute: 

    bash$ g++ -Wall -std=c++11 -L../lib/ -I../inc/ -o test.e test.cpp -liers2010

    In case you don't have g++, either install it or adapt the previous line to your prefered compiler.

  3. The previous step should have created the executable test.e. Run this program to test the installation of the library. The output should be self-explanatory. You can also save the output in a file for further use (see).
