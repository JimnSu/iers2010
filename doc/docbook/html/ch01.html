Introduction

The libiers10++ library includes a variety of functions, implementing the models defined by IERS in technical note 36, IERS Conventions (2010). The document is available in the IERS website.

The IERS, has a dedicated website, where official implementations of these models are distributed, via source code in FORTRAN programing language. Updates can be found here. The source code and documents can be downloaded and used, following the terms and conditions defined therein.

libiers10++ was developed in Dionysos Satellite Observatory of the National Technical University of Athens, to assist its own geodesy-oriented scientific activities. To that end, most of the IERS routines were translated and compiled into a library for further use. The main programing language used in the Laboratory is C++, consequently, the library is written in this language.

Source Code

The languge of choice for the source code is Standard C++. The current ISO C++ standard, known as C++11, is used.

Note

The library uses a number of C++11 features, so make sure you use a compiler with support for this ISO version.
