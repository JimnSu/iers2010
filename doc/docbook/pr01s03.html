Contributions

Most of the source code accompaning libiers10++ is translated from FORTRAN routines and/or subroutines provided by the International Earth Rotation and Reference Systems Service (IERS). Although we have tried to 'stick' as much as possible to the original implementation, there are always minor or major modιfications. The results of any function though, should coincide exactly with the ones obtained when using the original FORTRAN routines (see ???)
